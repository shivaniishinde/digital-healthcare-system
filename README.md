# Digital Healthcare System

A backend healthcare management system built using Spring Boot.

## Tech Stack

* Java 17
* Spring Boot 3.5.15
* Spring Data JPA
* MySQL
* Spring Security
* Lombok

## Features

* User Management
* Doctor Management
* Patient Management
* Appointment Management
* Prescription Management
* Role-Based Access Control
